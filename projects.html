Carte Interactive des Villes de France

React Spring Boot MySQL Docker API REST

Application web full-stack permettant d'explorer les villes françaises à travers une interface cartographique interactive. Ce projet illustre une architecture microservices complète avec séparation claire des responsabilités.

Fonctionnalités principales

Les utilisateurs peuvent cliquer n'importe où sur la carte de France métropolitaine pour découvrir les villes à proximité. L'application interroge une base de données contenant les informations géographiques et démographiques des communes françaises, avec des paramètres de recherche configurables.

Architecture technique

Frontend : Développé en React pour une expérience utilisateur dynamique, avec gestion des interactions cartographiques en temps réel.

Backend : API REST en Spring Boot (Java) gérant les requêtes géospatiales et la logique métier.

Base de données : MySQL optimisée pour les requêtes géographiques avec indexation des coordonnées.

Déploiement : Architecture containerisée avec Docker Compose pour un déploiement simplifié.

Tests et qualité

Implémentation de tests unitaires (JUnit/Mockito) pour le backend et tests d'acceptation (Cypress) pour le frontend, garantissant la robustesse de l'application.

Refonte UX du Site E-commerce "La Belle Vie"

UX/UI Design Figma Tests Utilisateurs Prototypage

Projet collaboratif de refonte complète de l'expérience utilisateur d'un site e-commerce, réalisé dans le cadre des cours UX Design à l'ESIEA. L'objectif était de résoudre les points de friction identifiés dans le parcours d'achat.

Analyse et recherche utilisateur

Conduite d'interviews utilisateurs pour identifier les frustrations principales : navigation peu intuitive, manque de filtres de recherche performants, et parcours d'achat complexe. Benchmark concurrentiel réalisé sur des références comme Carrefour et Franprix.

Conception et prototypage

Création d'une maquette interactive sur Figma intégrant les améliorations identifiées : système de filtrage avancé, navigation optimisée par catégories, et tunnel de commande simplifié avec confirmations visuelles.

Scénario de test validé

Développement d'un parcours utilisateur complet depuis l'arrivée sur la page d'accueil jusqu'à la confirmation de commande, incluant l'exploration des catégories, le filtrage (ex: Viande > Boucherie > Bœuf), l'ajout au panier avec animations, et le processus de paiement sécurisé.

Résultats

Les tests utilisateurs ont validé une amélioration significative de la fluidité du parcours et une réduction des abandons de panier grâce aux nouvelles fonctionnalités proposées.
